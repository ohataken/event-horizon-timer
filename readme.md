# Event Horizon Timer

## A Timer for People Sharing the Same Moment

**Event Horizon Timer** is a web-based timer app designed for events, presentations, and group activities.  
Only the host can control the timer, while all participants see the same countdown in real-time.

Whether you're leading a workshop or giving a talk,  
make sure everyone is on the same rhythm, heading toward the same goal—together.

---

## Features

- **Real-Time Synchronization**  
  Any timer updates are instantly reflected across all connected devices.

- **Separation of Control and View**  
  Only the facilitator operates the timer; participants simply follow it.

- **No Installation Needed**  
  Just open your browser. It works anywhere, anytime.

- **Multiple Timers Support**  
  Perfect for hackathons or events with parallel sessions.
